BẠN LO LẮNG VỀ SỰ PHÙ HỢP CỦA KHÓA HỌC NÀY?

Hãy an tâm nếu bạn hiện đang là:

  • Sinh viên các trường Đại học, Cao đẳng, Trung cấp nghề chưa có kinh nghiệm thực tế, đang cần trang bị và hoàn thiện một cách có hệ thống kiến thức về Thiết kế Web.
  • Người đi làm cần bổ sung, củng cố và chuẩn hóa kiến thức để có thể tự tay chỉnh sửa giao diện Web, tăng khả năng thăng tiến trong nghề nghiệp
  • Các bạn yêu thích thiết kế Website và mong muốn một cơ hội nghề nghiệp, thay đổi ngành nghề,…

Thiết kế giao diện Web UI/UX:

  • Các công cụ thiết kế cơ bản thiết kế layout
  • Thiết kế banner web đa kích thước
  • Ứng dụng Grid Layout thiết kế khung trang web
  • Responsive Layout Web với 3 kích thước thiết bị phổ biến
  • Landing Page: Bố cục trang Landing Page
  • Ứng dụng UI/UX trong thiết kế Header Sider
  • Màu sắc/ Hình ảnh website:Ứng dụng UI/UX thiết kế Menu, navigation và footer
  • Font chữ cho web: Ứng dụng UI/UX Ecomerce / Blog & Magazine,...
  • Ứng dụng UI/UX trên thành phần web: form, media...
  • Thiết kế layout Responsive web: Porfilio cá nhân ứng dụng UI/UX
  • Thủ thuật tối ưu hóa hình ảnh Web và thủ thuật xuất ảnh cho website

Phát triển Website với Dreamweaver:

  • Ngôn ngữ HTML
  • Thiết kế, trình bày trang web
  • Xây dựng nội dung trang web
  • CSS-Cascading Style Sheet
  • Sử dụng Behavior cho các đối tượng trong trang web
  • Thiết kế biểu mẫu-FORM
  • Xây dựng trang mẫu - Template
  • Slide - Tạo nhanh Template
  • Tự thiết kế website đơn giản
  • Xây dựng trang khung - Framesets

Thiết kế web động với Animtion

  • Tổng quan Animation
  • Kết hợp hiệu ứng cho đối tượng
  • Hiệu ứng chuyển trang
  • Thiết kế Banner
  • Các dạng Banner
  • Diễn hoạt Animation nâng cao

Thiết kế Web cho thiết bị di động (Responsive Web với Bootstrap)

  • Khái quất về Responsive (bố cục Responsive, phân tích Responsive chuẩn, cấu trúc CSS trong Responsive)
  • Xây dựng trang Responsive cơ bản
  • Hệ thống lưới (Grid) & ứng dụng xây dựng Responsive Web với Bootstrap theo dạng lưới.
  • CSS3 Frarework - Ứng dụng Framework dụng Responsive
  • Ứng dụng Layout cá nhân tương thích Responsive

Thiết kế Web với Wordpress + Đồ án môn hình

  • Giới thiệu và cài đặt WordPress (cài đặt locolhost, cấu hình MySQL, Cài đặt WordPress)
  • Tạo bài viết (post) trong WordPress
  • Quản lí và thiết bị bình luận
  • Chèn Video, hình ảnh và WordPress
  • Tạo page
  • Cài đặt Themes
  • Cài đặt Plugin
  • Quản lý và thiết lập Menu
  • Sử dụng Widget
  • Tùy biến quyền người dùng
